# @mcp-india/zoho-crm

## 0.1.0

### Features

- 14 MCP tools: contacts, deals, tasks, calls, notes, pipeline summary, revenue forecast
- OAuth2 client with automatic token refresh and thundering herd prevention
- Rate limiting (10 req/sec) via shared RateLimiter
- Multi-region support (US, India, EU) via ZOHO_API_DOMAIN
- Full TDD and BDD test coverage
- Zero-install usage via `npx -y @mcp-india/zoho-crm`
