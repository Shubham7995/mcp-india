# @mcp-india/gst-india

## 0.1.0

### Features

- 9 fully offline MCP tools: GSTIN validation, invoice number validation, state info, GST calculation, supply type determination, reverse calculation, HSN/SAC search, HSN/SAC detail lookup, and rate slab listing
- Zero configuration required — no API keys, no environment variables
- Bundled static data: 500+ HSN codes, 120+ SAC codes, 38 state/UT codes, all rate slabs
- Full TDD and BDD test coverage
- Zero-install usage via `npx -y @mcp-india/gst-india`
