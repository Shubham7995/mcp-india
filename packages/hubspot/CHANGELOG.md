# @mcp-india/hubspot

## 0.1.0

### Minor Changes

- Initial release of HubSpot CRM MCP server
- 18 tools across 5 domains:
  - **Contacts**: search, get, create, update contacts
  - **Companies**: search, get, create, update companies
  - **Deals**: search, get, create, update deals
  - **Engagements**: create notes, tasks, log emails, list activities
  - **Reports**: pipeline summary by stage, deal forecast with weighted values
- Native fetch client with Bearer token auth (no heavy SDK)
- Rate limiting at 10 req/s
